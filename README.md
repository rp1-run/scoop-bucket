# rp1 Scoop Bucket

Official Scoop bucket for the rp1 CLI on Windows.

## Installation

```powershell
scoop bucket add rp1 https://github.com/rp1-run/scoop-bucket
scoop install rp1
```

## Updating

```powershell
scoop update rp1
```

## About

rp1 provides AI-assisted development workflows for Claude Code and OpenCode.
